The Emergency Communication System is a web-based platform designed to assist Indigenous communities in Wilcannia, Australia. It connects residents with hospitals and water suppliers to provide timely emergency alerts.  

The system integrates **GPS tracking** and **voice-based reporting** to ensure real-time communication even in low-connectivity environments. The interface is simple and accessible, making it easy for users of all digital literacy levels to report emergencies and receive assistance.
